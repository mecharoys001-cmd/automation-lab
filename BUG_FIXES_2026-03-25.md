### 5. reczKKUMYyG058zk1 — Admin Remove still hangs
**Status:** ✅ FIXED

**Problem:**
Waiting for API call and subsequent refetch caused perceived "hang" in the UI, especially on slow connections.

**Solution:**
Implemented optimistic update pattern:
1. Immediately remove user from UI
2. Close modal instantly
3. Make DELETE request in background
4. Show temporary "Removing..." toast
5. On success: update toast to "removed"
6. On failure: rollback by refetching and show error

**Benefits:**
- UI never hangs or appears frozen
- Immediate feedback to user
- Graceful error handling with rollback

**Files Modified:**
- `app/tools/scheduler/admin/roles/page.tsx` - Rewrote `handleRemove` function with optimistic updates
